我对项目做了以下改造：
1. 删除多用户支持
2. 工作区改为跟随当前目录（`process.cwd()`），不再硬编码到 `~/.cc-weixin/workspace/`
3. 添加微信发送/resume和/new指令的支持, 更方便会话管理：每次运行cc-weixn，默认新开会话，可通过微信发送/resume来恢复旧会话，在旧会话里通过微信发送/new新开会话
4. 开放所有权限给Claude Code，彻底跳过所有权限检查

改造原因：
1. 我只有一个微信，我不需要多用户功能，所以做了第一个改造
